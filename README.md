# homebrew-tap

Homebrew formulae for [Praxis](https://praxis-ai.io) — the AI-native development environment.

## Install

```sh
brew install systimus/tap/praxis-proxy
```

After install, configure your API key and per-tool setup at the
[Praxis setup page](https://praxis-ai.io/setup) for full onboarding
(API key, per-tool config, test connection).

## What's here

| Formula | Description |
|---------|-------------|
| `praxis-proxy` | Local HTTPS passthrough for Claude Code that captures exact AI spend |
